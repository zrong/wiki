Lesson 1
===============

陈述句的句子成分：

+-------+--------+--------+-------------+------+--------+-------+
| 6     | 1      | 2      | 3           | 4    | 5      | 6     |
+-------+--------+--------+-------------+------+--------+-------+
|       | Who?   |        | SomeoneWho? | How? | Where? | When? |
| When? | Which? | Action | Which?      |      |        |       |
|       | What?  |        | What?       |      |        |       |
+-------+--------+--------+-------------+------+--------+-------+


其中表示时间的词可以放在开头，也可以放在句子的结尾。

- 1 为主语，一般为名词或者物主代词。
- 2 为谓语，一般使用系动词或者实意动词。
- 3 为宾语，前面可以加上定语。
- 4 为状语
- 5 为补语
- 6 为时间状语。
