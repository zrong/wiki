Lesson 1
===============

不关你的事 ::

    none of your business.

none 相当于 ::

    none = not any = no one

none of 这个短语的语气比较强烈，表达断然的口气 ::

    None of your silly remarks!

某人的事，某人关心的事，某人的分内事 ::

    sb. 's business.


去看戏 ::

    go to the theatre = go to the theatre to see a play

通常不使用定冠词的例子 ::

    go to school

    go to bed

    go to church
